# RYNC DRAW — موقع جاهز للنشر

## المحتوى
- `index.html` · `style.css` · `script.js`
- `logo.png` · `vercel.json`

## النشر على Vercel
1. ارفع المجلد كاملاً إلى مستودع (GitHub/GitLab) أو اسحبه إلى vercel.com.
2. لا حاجة لأي إعدادات. اضغط **Deploy**.
3. الكاميرا تعمل تلقائياً لأن Vercel يخدم HTTPS.

## الإشارات
- ☝️ السبابة فقط → ارسم
- ✌️ سبابة + وسطى → امسح المنطقة
- 👌 قرصة → حرّك قطعة واحدة
- ✊ قبضة → حرّك كل الرسم
- 👋 يد مفتوحة (يمين/يسار) → غيّر اللون
- 🫰 فقشة → مسح الكل
- 👎 إبهام للأسفل → تراجع

## اختصارات لوحة المفاتيح
`C` مسح · `Z` تراجع · `مسافة` تبديل الوضع
